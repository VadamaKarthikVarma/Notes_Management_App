üß† Brief Description

The Notes Management System is a full-stack web application designed to help users securely manage their personal notes online. Built using Flask (Python) for the backend and MySQL for database management, this system enables users to create, read, update, and delete (CRUD) their notes efficiently.

Each registered user has a private workspace, ensuring complete data privacy. The application supports user registration, login, and logout functionalities with password encryption for enhanced security. A built-in search feature allows users to quickly find notes based on titles or content, making it a practical and efficient digital note-keeping platform.
